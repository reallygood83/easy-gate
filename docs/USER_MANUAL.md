## 소개

**Easy Gate**는 옵시디언에서 웹사이트를 임베드하고, AI를 활용하여 웹 콘텐츠를 분석하고 저장할 수 있는 강력한 플러그인입니다.

### 주요 기능

- **멀티탭 웹 브라우징**: 옵시디언 내에서 여러 웹사이트를 탭으로 관리
- **웹페이지 저장**: 전체 페이지 또는 선택 영역을 마크다운 노트로 저장
- **AI 분석**: OpenAI, Anthropic, Google Gemini 등 다양한 AI를 활용한 콘텐츠 분석
- **YAML Properties 자동 생성**: 모든 저장 노트에 메타데이터 자동 포함

---

## 설치 방법

### 방법 1: 옵시디언 커뮤니티 플러그인

1. 옵시디언 설정 (⚙️) 열기
2. **커뮤니티 플러그인** → **찾아보기** 클릭
3. "Easy Gate" 검색
4. **설치** 버튼 클릭
5. **활성화** 버튼 클릭

### 방법 2: 수동 설치

1. GitHub에서 최신 릴리즈 다운로드
2. `main.js`, `manifest.json`, `styles.css` 파일을
   `YOUR_VAULT/.obsidian/plugins/easy-gate/` 폴더에 복사
3. 옵시디언 재시작
4. 설정에서 플러그인 활성화

## 기본 사용법

### Gate(게이트) 생성하기

1. 좌측 리본에서 **지구본 아이콘** 클릭
2. 또는 명령어 팔레트 (Ctrl/Cmd + P) → "Easy Gate: Open Gate Manager"
3. URL 입력 후 Enter

### 탭 관리

- **탭 전환**: 상단 탭바에서 원하는 사이트 클릭
- **새 탭 추가**: 주소창에 URL 입력 후 Enter
- **탭 삭제**: 탭의 X 버튼 클릭 (확인 대화상자 표시)

### 네비게이션

- **뒤로/앞으로**: 좌측 화살표 버튼 사용
- **새로고침**: 새로고침 버튼 클릭
- **홈으로**: 홈 버튼 클릭

## 웹페이지 저장 기능

### 📥 저장 버튼

툴바의 **📥** 버튼을 사용하여 웹페이지를 옵시디언 노트로 저장합니다.

#### 원클릭 저장 (📥 버튼 직접 클릭)
- 현재 보고 있는 웹페이지 전체를 즉시 마크다운 노트로 저장

#### 저장 옵션 (▼ 드롭다운)

| 옵션 | 설명 |
|------|------|
| **📄 전체 페이지 저장** | 웹페이지 전체 콘텐츠를 새 노트로 저장 |
| **✂️ 선택 영역 저장** | 웹페이지에서 드래그로 선택한 텍스트만 저장 |
| **📎 기존 노트에 추가** | 저장된 기존 노트에 콘텐츠 추가 |
| **⚙️ 저장 설정** | 기본 저장 폴더, 파일명 형식 등 설정 |

### 저장된 노트 형식

모든 저장 노트는 **YAML frontmatter**가 자동 포함됩니다:

```yaml
---
source: https://example.com/article
clipped: 2025-12-18
author: John Doe
site: Example Site
---
```

## AI 분석 기능

### 🤖 AI 버튼

툴바의 **🤖** 버튼을 사용하여 AI 기반 콘텐츠 분석을 수행합니다.

> **참고**: AI 기능 사용을 위해서는 API 키 설정이 필요합니다.

#### 원클릭 분석 (🤖 버튼 직접 클릭)
- 현재 페이지 전체를 AI가 분석하여 요약 노트 생성

#### AI 분석 옵션 (▼ 드롭다운)

| 옵션 | 설명 |
|------|------|
| **📄 전체 페이지 분석** | 웹페이지 전체 콘텐츠를 AI로 분석 |
| **✂️ 선택 영역 분석** | 선택한 텍스트만 AI로 분석 |
| **📋 템플릿 사용** | 미리 정의된 분석 템플릿 사용 |
| **🔍 분석 모달 열기** | 상세 분석 옵션을 선택할 수 있는 모달 |
| **⚙️ API 키 설정** | AI 제공자별 API 키 설정 |

### 분석 템플릿

| 템플릿 | 설명 |
|--------|------|
| **📋 기본 요약** | 핵심 내용을 간결하게 요약 |
| **📚 학습 노트** | 학습에 최적화된 형식으로 정리 |
| **📊 분석 리포트** | 상세 분석 리포트 생성 |
| **💡 아이디어 노트** | 창의적 아이디어와 인사이트 추출 |

### AI 분석 결과 노트

AI 분석 결과는 **완전한 YAML frontmatter**와 함께 저장됩니다:

```yaml
---
title: "기사 제목 - AI 분석"
source: "https://example.com/article"
created: 2025-12-18
type: ai-analysis
provider: openai
template: basic-summary
tags:
  - ai-analysis
  - easy-gate
---
```

## 설정

### 플러그인 설정 열기

1. 옵시디언 설정 (⚙️) 열기
2. 좌측 메뉴에서 **Easy Gate** 클릭

### 저장 설정 (Clipping Settings)

| 설정 | 설명 |
|------|------|
| **기본 폴더** | 저장된 노트가 생성될 폴더 |
| **파일명 형식** | 노트 파일명 패턴 ({title}, {date}, {site} 사용 가능) |
| **URL 포함** | YAML에 원본 URL 포함 여부 |
| **날짜 포함** | YAML에 저장 날짜 포함 여부 |
| **작성자 포함** | YAML에 작성자 정보 포함 여부 |
| **HTML 포함** | 원본 HTML 코드 포함 여부 |

### AI 설정

| 설정 | 설명 |
|------|------|
| **AI 제공자** | 사용할 AI 서비스 선택 |
| **API 키** | 선택한 AI 제공자의 API 키 |
| **기본 언어** | AI 응답 언어 (기본: 한국어) |
| **모델** | 사용할 AI 모델 선택 |

### 지원되는 AI 제공자

| 제공자 | 설명 |
|--------|------|
| **OpenAI** | GPT-4, GPT-3.5-turbo |
| **Anthropic** | Claude 3 Opus, Sonnet, Haiku |
| **Google** | Gemini Pro |
| **Grok** | xAI의 Grok 모델 |
| **GLM** | Zhipu AI의 GLM 모델 |

## FAQ

### Q: 저장 버튼(📥)이나 AI 버튼(🤖)이 보이지 않아요.

**A**: Desktop 앱에서만 이 기능들이 활성화됩니다. 모바일 앱에서는 iframe 제한으로 인해 해당 기능들이 비활성화됩니다.

### Q: AI 분석이 작동하지 않아요.

**A**: 다음을 확인해주세요:
1. 설정에서 AI API 키가 올바르게 입력되었는지 확인
2. API 키의 유효성 확인 (만료 여부, 사용량 제한 등)
3. 인터넷 연결 상태 확인

### Q: "선택 영역 저장"이 작동하지 않아요.

**A**: 먼저 웹페이지에서 텍스트를 드래그하여 선택한 후, "✂️ 선택 영역 저장"을 클릭해주세요. 선택된 텍스트가 없으면 오류 메시지가 표시됩니다.

### Q: YAML Properties가 생성되지 않아요.

**A**: Easy Gate v2.0.4 이상에서는 모든 저장 및 AI 분석 노트에 YAML frontmatter가 자동 포함됩니다. 이전 버전을 사용 중이라면 업데이트해주세요.

### Q: 특정 웹사이트가 로드되지 않아요.

**A**: 일부 웹사이트는 iframe/webview 임베딩을 차단합니다. 다음을 시도해보세요:
1. 다른 Gate에서 동일 URL 접근
2. User-Agent 설정 변경 (Gate 설정에서)
3. 프로필 키 변경 (세션 격리)

### Q: 게이트를 삭제했는데 복구할 수 있나요?

**A**: 게이트 삭제 시 확인 대화상자가 표시됩니다. 삭제를 확인하면 복구가 불가능하므로 신중하게 진행해주세요. 저장된 노트는 삭제되지 않습니다.
